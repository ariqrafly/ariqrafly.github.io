Broadcasting Simulator in VR

Developing a VR game that serves as a simplified broadcast studio simulator. This project is built on Unity Engine. The goal of this project is to provide an overview of the simulation for Video Technicians, Audio Mixers, Lighting Technicians, and Field Directors during the broadcast or recording process.
